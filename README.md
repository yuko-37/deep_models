### Description
DeepModels is a study repository containing practical examples of building deep learning models from scratch and with TensorFlow/Keras.

It includes fully connected neural networks, convolutional neural networks (CNNs), recurrent neural networks (RNNs), and experiments with CNN transfer learning. Some models are implemented from scratch using NumPy to understand core mechanics, while others leverage TensorFlow/Keras for modern deep learning workflows.

The repository also explores key techniques such as dropout, regularization, and different weight initializers to understand their effect on model training and performance.

The goal is to explore different architectures, experiment with hyperparameters, and gain hands-on experience with deep learning concepts and techniques.

### Conda commands:
```bash
conda env create -f environment.yml
conda activate deep
conda env update --name deep --file=./environment.yml --prune
```
